# Weather Agent using Streamlit and OpenWeatherMap API

This project is a Streamlit-based web application that retrieves and displays real-time weather information for a given city using the OpenWeatherMap API.

The application provides a simple and interactive interface where users can enter a city name and get current weather conditions instantly.

---

## Project Overview

The Weather Agent fetches real-time weather data using an external API and displays the results in a user-friendly format.

Workflow:

1. User enters a city name.
2. The application sends a request to the OpenWeatherMap API.
3. Weather data is retrieved and processed.
4. The result is displayed on the Streamlit interface.

---

## Features

- City-based weather search
- Real-time weather data retrieval
- Error handling for invalid cities
- Loading spinner for better user experience
- Clean and centered Streamlit layout

---

## Technologies Used

- Python
- Streamlit
- OpenWeatherMap API
- Requests (inside weather_agent module)
